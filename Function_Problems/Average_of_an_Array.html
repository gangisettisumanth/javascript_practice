//8. **Average of an Array**: Write a function that calculates the average of the numbers in an array. function arrayAverage(arr) { if (arr.length === 0) { return 0; } const sum = arr.reduce((accumulator, currentValue) => accumulator + currentValue, 0); return sum / arr.length; } const numbers = [1, 2, 3, 4, 5]; console.log(arrayAverage(numbers));
